Updog

Updog is a digital journaling app I created for a User Interfaces class. Users can track and reflect on their innermost thoughts and feelings, and each journal entry is encrypted. I developed it using the Flutter framework and Dart language, and I designed the assets using Pixilart.

Click here to see the Github repo!

Home IDIR Lab

I built a home Intrusion Detection and Incident Response (IDIR) lab to simulate real-world cybersecurity environments. The lab runs six virtual machines: a PfSense VM for firewall and routing capabilities, a SIEM VM using Splunk for security event management, an IPS VM running Suricata for network traffic analysis and intrusion prevention, a Kali Linux VM for penetration testing, and a Metasploitable 2 VM as a vulnerable target machine.

Sploosh!

Sploosh is a hydration-tracking app I designed and developed with a team for an Application Programming class. It gamifies the act of drinking water by rewarding users for their progress towards their daily hydration goal. Through Sploosh, users can build consistent hydration habits, and care for a virtual fish in the process.

We used Java, XML, and Android Studio to develop this app. This project gave us the opportunity to learn a lot about app development and user experience design.
